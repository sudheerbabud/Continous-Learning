Introduction of MySQL

MySQL is a relational database management system used for manipulating data.

SQL Basics

INSERT INTO table_name (column1, column2,...) VALUES (value1, value2,...);

CREATE TABLE example_table (
id INT AUTO_INCREMENT PRIMARY KEY, -- Primary key with auto-increment
name VARCHAR(100) NOT NULL,       -- NOT NULL constraint
age INT CHECK (age >= 18),        -- CHECK constraint
email VARCHAR(255) UNIQUE,        -- UNIQUE constraint
customer_id INT,                  -- Foreign key
FOREIGN KEY (customer_id) REFERENCES customers(customer_id), -- Foreign key reference
created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP, -- DEFAULT with current timestamp
status VARCHAR(50) DEFAULT 'active', -- DEFAULT with a static value
description TEXT DEFAULT NULL,  -- DEFAULT NULL
INDEX idx_name (name)            -- CREATE INDEX
);

Normalization

  • 1NF (First Normal Form)
    • the table must have only one row per unique combination of values
    • the rows must contains only single value
  • 2NF (Second Normal Form)
    • It must be in 1NF
    • All the non-prime attributes must depend on the prime attribute (primary key). It is partial dependency
  • 3NF (Third Normal Form)
    • It must be in 2NF
    • All the non-prime attributes depends on non- prime attribute and the prime. It is Transitive dependency like A --> B --> C. that means A -->C
  • BCNF (Boyce-Codd Normal Form)
    • It must be in 3NF
    • It is Functional Dependency X(Y)
  • 4NF (Fourth Normal Form)
    • It must be in 3NF
    • Eliminate multi-valued dependencies by separating them into distinct tables.
  • 5NF (Fifth Normal Form)
    • It must be in 4NF
    • It is free from join dependencies.

Joins

Used to joins two or more tables

  1. INNER JOIN

    2. SELECT column_names FROM table1 INNER JOIN table2 ON table1.column_name = table2.column_name

  2. LEFT JOIN

    3. SELECT column_names FROM table1 LEFT JOIN table2 ON table1.column_name = table2.column_name

  3. RIGHT JOIN

    4. SELECT column_names FROM table1 RIGHT JOIN table2 ON table1.column_name = table2.column_name

  4. FULL JOIN

    5. SELECT column_names FROM table1 FULL JOIN table2 ON table1.column_name = table2.column_name

  5. CROSS JOIN

    6. SELECT column_name FROM table1 CROSS JOIN table2

  6. NATURAL JOIN

    7. SELECT column_name FROM table1 NATURAL JOIN table2

  7. SELF JOIN

    8. SELECT A.column_name, B.column_name FROM table1 A, table2 B ON A.column_name = B.column_name

Set Operations

  • UNION

    • SELECT * FROM table1_name UNION (SELECT * FROM table2_name) --> adds two tables without duplicates

  • UNION ALL

    • SELECT * FROM table1_name UNION ALL (SELECT * FROM table2_name) --> adds two tables with duplicates

  • INTERSECT

    • SELECT * FROM table1_name INTERSECT (SELECT * FROM table2_name) --> adds two tables without duplicates

  • MINUS

    • SELECT * FROM table1_name MINUS (SELECT * FROM table2_name) --> removes duplicates from table2 _name refering to table1_name
