Introduction of HTML

HTML --> HyperText Markup Language --> It is used to create the structure of the web page.

HTML is not a programming language, it is a markup language. it won't gives errors if you write it in a wrong way.

HTML Tags - 1

  • DOCTYPE HTML --> it shows the document type of the HTML document.
  • HTML --> It is the root element of an HTML document.
  • Head --> It contains meta-information about the HTML document.
  • title --> It is used to give the title of the HTML document.
  • Body --> It contains the main content of the HTML document.
  • header --> It is used to give the header of the HTML document.
  • footer --> It is used to give the footer of the HTML document.
  • style --> It is used to give the Internal styling of the HTML document.
  • script --> It is used to give the Internal script in the HTML document.
  • comment --> It is used to give the comments in the HTML document.

Semantic Tags

Sematic tags are the tags that we can know the purpose of the tag by seeing the name it self.

  • header - It is used to define the header of the tag.
  • footer - It is used to define the footer of the tag.
  • nav - It is used to define the navigation bar.
  • section - It is used to define a section in a document.
  • article - It is used to define a self-contained composition in a document.
  • aside - It is used to define content that is tangentially related to the content around it.
  • figure - It is used to define self-contained content.
  • figcaption - It is used to define a caption for a element.
  • main - It is used to define the main content of a document.
  • time - It is used to define a specific time or date.
  • mark - It is used to define a marked or highlighted part of a text.
  • progress - It is used to show the progress of a task.

NonSemantic Tags

NonSematic tags are the tags that we can't know the purpose of the tag by seeing the name it self.

  • div - It is a generic container for flow content.
  • span - It is a generic inline container for flow content.
  • b - represents text bold but it won't gives the idea about the text
  • br - It is used to insert a line break.
  • img - It is used to embed images.
  • i- represents text italic but it won't gives the idea about the text
  • u - represents text underline but it won't gives the idea about the text
