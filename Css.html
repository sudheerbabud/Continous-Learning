Introduction of CSS

CSS - Cascading Style Sheets - Used to style HTML documents

We can apply style for HTML elements in 3 ways

  1. Inline Styles - We can apply style directly to the HTML element using style attribute
  2. Internal Stylesheets - We can apply style to the HTML document using style tag inside the head tag
  3. External Stylesheets - We can apply style to the HTML document using link tag inside the head tag

We can apply style to the HTML elements using CSS selectors

  • Tag Name - Selects all HTML elements with the given tag name
  • Class - Selects HTML elements with the given class name
  • ID - Selects the HTML element with the given ID
  • Universal Selector - Selects all HTML elements
  • Attribute Selector - Selects HTML elements with the specified attribute
  • Pseudo-class Selector - Selects HTML elements based on their state
  • Group Selector - Selects multiple HTML elements
  • Descendant Selector - Selects HTML elements that are descendants of a specified element
  • Child Selector - Selects HTML elements that are direct children of a specified element
  • Adjacent Sibling Selector - Selects HTML elements that are adjacent siblings

Text Properties

  1. font-family - Specifies the font family for text
  2. font-size - Specifies the size of the font
  3. font-weight - Specifies the weight of the font
  4. color - Specifies the color of the text
  5. text-align - Specifies the horizontal alignment of the text
  6. text-decoration - Specifies the decoration of the text
  7. text-transform - Specifies the capitalization of the text
  8. line-height - Specifies the line height
  9. text-indent - Specifies the indentation of the first line of text
  10. text-shadow - Specifies a shadow effect on the text
  11. word-spacing - Specifies the spacing between words

Basic Properties

  • width - Specifies the width of an element
  • height - Specifies the height of an element
  • border - Specifies the width, style, and color of an element's border
  • padding - Specifies the width, style, and color of an element's padding
  • margin - Specifies the width, style, and color of an element's margin
  • color - color of the text
  • background-color - color of the background
  • border-radius - Specifies the rounded corners of an element

Box Model

  • Content Area - The area inside the padding and border
  • Padding - The area inside the content area, between the content and the border
  • Border - The area around the padding and content area
  • Margin - The area outside the border, between the border and the next element

Units in CSS

  • px - pixels
  • em - relative to the font-size of the parent element
  • rem - relative to the root element (html)
  • vw - relative to the viewport width
  • vh - relative to the viewport height
  • cm - centimeters
  • mm - millimeters
  • in - inches
  • pt - points
  • pc - picas
  • % - percentage

Position in CSS

  • static - The element is positioned relative to the normal document flow
  • relative - The element is positioned relative to its normal position
  • absolute - The element is positioned relative to the nearest positioned ancestor
  • fixed - The element is positioned relative to the viewport
  • sticky - The element is positioned relative to the nearest scrollable ancestor
  • inherit - The element inherits the position from its parent element

Media Queries

We can use media queries to apply different styles based on the screen size or device type.

@Media(width:600px){
div{
    background-color: blue;
}
}

Animation

Animation - Used to create animations in CSS

@keyframes animation{
0% {background-color: red;}
50% {background-color: yellow;}
100% {background-color: green;}
}

Transition

  • transition-property - font-size, width..
  • transition-duration - time 2s/4ms
  • transition-timing-function - linear, ease, ease-in, ease-out, ease-in-out
  • transition-delay - time 1s/2ms
  • transition-fill-mode - forwards, backwards, both

Transform

  • transform - scale(2), rotate(45deg), translate(100px, 100px)
  • transform-origin - top left, bottom right, center center
  • transform-style - preserve-3d, flat, preserve-3d
  • transform-box - border-box, content-box, fill-box
  • transform-scale - scale(2), scale(2, 3)
  • transform-rotate - rotate(45deg)
  • transform-translate - translate(100px, 100px)
  • transform-skew - skew(45deg)
