Introduction of Bootstrap

Bootstrap - Bootstrap is a free and open-source front-end framework designed to simplify web development. It provides a collection of tools, including CSS, JavaScript, and HTML templates, to help developers build responsive, mobile-first websites quickly and efficiently.


Key features of Bootstrap include:

  1. Grid System:A flexible, 12-column grid system for building layouts that adapt to different screen sizes.
  2. Predefined CSS Classes: Includes styles for typography, buttons, forms, navigation bars, and other common UI components.
  3. Responsive Design: Built-in media queries make websites responsive, automatically adjusting to different devices like smartphones, tablets, and desktops.
  4. cJavaScript Plugins:Includes interactive elements like carousels, modals, and tooltips that can be easily integrated using jQuery.
  5. Customization:You can customize the look and feel by adjusting the source code or using Bootstrap's theming options.
Versions of Bootstrap

Bootstrap has been developed and released by Twitter, Inc. The latest stable version is Bootstrap 5.1.0.

Key Differences between 3,4,5:

  1. Grid System: Bootstrap 3 uses a traditional grid system, while Bootstrap 4 and 5 utilize flexbox for more flexible layout options.
  2. Components: Bootstrap 4 introduced "cards" as a primary layout element, replacing the older "panels" from Bootstrap 3.
  3. JavaScript Dependency: Bootstrap 3 and 4 rely on jQuery, while Bootstrap 5 uses only native JavaScript.
  4. Customization: Bootstrap 4 and 5 offer greater customization potential with Sass support.
  5. Browser Support: Bootstrap 5 may not support older browsers like Internet Explorer 11, which Bootstrap 3 and 4 might still support.

Grid System

Bootstrap's grid system uses a 12-column grid system, which means it can create layouts with equal width columns on different screen sizes. The grid system is responsive, meaning it adjusts the number of columns and their widths based on the device's screen size.
